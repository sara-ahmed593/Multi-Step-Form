## Description

Create a multi-step registration that allow users to submit personal information, select preferences, review their details, and complete the registration process through a series  of steps.
## Features

- The form  have 4 steps, with a progress indicator to show users their progress.
- Each step should have its own set of input fields.
- Users  able to navigate between steps using buttons.
- The form  have validation to ensure that all required fields are filled out before the user can proceed to the next step.
- The form  have a summary page at the end, where users can review their inputs before submitting.
- After submitting the form, users can see a congratulatory message .
